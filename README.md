## 📌 Muammo

O'zbekistonda shaharlararo yirik yuklar uchun raqamli yetkazib berish xizmati mavjud emas. BTS va Yandex Delivery kabi xizmatlar faqat kichik jo'natmalarni qabul qiladi. Mebel, maishiy texnika, ulgurji tovarlar yoki yangi gullar yuborish uchun shaxsiy aloqalarga yoki tasodifiy haydovchilarga murojaat qilish kerak.

**YukTaxi** bu muammoni hal qiladi — bo'sh yuk mashinalarini ularga to'g'ri keluvchi yuklarni yubormoqchi bo'lgan mijozlar bilan bog'laydi.

## 💡 Bizning ustunlik

### ⚡ AI Narx Bahosi
Yo'nalish, og'irlik, hajm va mavsum omillarini hisobga olgan holda adolatli narx oralig'ini real vaqtda hisoblaydi.

### 🛡️ Yuk Tekshiruvi™
Platformaning asosiy raqobat ustunligi:
- Mijoz yukni topshirishdan oldin 4 ta surat yuklaydi
- Haydovchi yukni olganda bir xil suratga oladi
- AI ikkalasini solishtiradi va mosligini tasdiqlaydi
- Bu ikkala tomonni ham noto'g'ri da'volardan himoya qiladi

### 🗺️ Jonli Kuzatuv
Yo'lda bo'lgan yukni real vaqtda kuzatish imkoniyati.

## 🚀 Ishga tushirish

```bash
# Repozitoriyni klonlash
git clone https://github.com/abdullohik/YukTaxi.git

# Papkaga kirish
cd YukTaxi

# Paketlarni o'rnatish
npm install

# Ishlab chiqish serverini ishga tushirish
npm run dev
```

Brauzerda `http://localhost:5173` manzilini oching.
